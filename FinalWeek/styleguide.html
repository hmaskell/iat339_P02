Fonts

Body Text

Interactive Elements

Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">DEFAULT BUTTON</a>

Rendered element:

DEFAULT BUTTON

In-text link

CSS selector:

a

Sample code:

<a href="#">Default Link</a>

Rendered element:

Default Link

Submit and Reset Buttons

CSS selector:

form, .button, button

Sample code:

<form method="post" action="#" id="contact">
<input type="submit" class="button" value="Submit">
<button class="button">Reset</button>
</form>

Rendered element:

Text Input Field

CSS selector:

form-search

Sample code:

<input type='text' id='form-search'/>

Rendered element:
